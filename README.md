# Pembelajaran Python - Kelas Terbuka

Repositori ini berisi sekumpulan file kode Python hasil pembelajaran mandiri yang bersumber dari channel YouTube Kelas Terbuka. Repositori ini dibuat sebagai dokumentasi pribadi untuk memahami konsep dasar hingga lanjutan dalam pemrograman Python.

## Struktur Folder 

```
├── README.md
├── exercise
│   └── file_latihan.py
└── src_materi
    └── file_materi.py
```

Semua materi disusun secara berurutan berdasarkan alur pembelajaran di Kelas Terbuka untuk mempermudah proses peninjauan kembali (review):

## Cara Menjalankan Kode

1. Pastikan Python sudah terinstal di sistem Anda. Cek menggunakan perintah:
   '''bash
   python --version
   ''' 

2. Clone repositori ini ke komputer lokal Anda:
   git clone https://github.com/RaffiN-07/python_learn-journey.git

3. Masuk ke dalam direktori/folder repositori:
   '''bash
   cd python_learn-journey
   '''

4. Jalankan salah satu file materi, misalnya:
   '''bash
   python main.py
   '''
